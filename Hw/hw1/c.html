An intermediate language that was designed towards system programming and is available on many playforms.

C++ was developed with the developer in mind, more or less being open to different programming styles.

Example of code
//Hello world
#include <iostream>
using namespace std;
int main{
cout << "Hello world;
return 0;
}

Pros
  • Popular and is widely supported.
  • Quick to compile.
  • Similar to other languages like Java, C, C#

Cons
  • No boundary checks, can be easy to corrupt data.
  • Not quite as flexible as languages like Java when it comes to object orientation.
  • Can sometimes be complicated to learn with the small standard library.
