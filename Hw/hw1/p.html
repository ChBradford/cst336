Python is a high-level langauge that focuses on being easy to read and is open source.

Example of code
#"Hello world"
hello = "Hello world"
print(hello)

Pros
  • Extremely easy to read, promoting the idea that is requires less thought and is easier to maintain.
  • Can do more with less code (no need to write a class to just print "hello world", like in java).

Cons
  • A bit on the slow side on execution.
  • Weak when it comes to mobile development.
  • Forced consistency on syntax (Somewhat subjective on whether it is a pro or con).
