Java is an object oriented programming language that is constructed to have as few dependencies as possible.

Java was built on C++, hence they look rather similar with a few differences.

Example of code
/*Hello world*/
public class HelloWorld{
public static void main(String[] args) {
System.out.println("Hello, World");
}
}

Pros
  • Java is great for object oriented programming (where classes contain variables and functions that work together to complete a program).
  • Java is cross platform, backed by the mantra of WORA (write once run anywhere).
  • Automatic memory management for garbage collection.

Cons
  • It can run a little slow at times, automatic garbage collection can impact performance at times.
  • Looks a little long at times, takes a LOT more code to print something simple like "Hello world" than it does in python or c++.
